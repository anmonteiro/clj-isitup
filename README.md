## Usage

For now (to check [Google](http://google.com)):

    $ lein trampoline run -- google.com

For multiple sites:

    $ lein trampoline run -- google.com facebook.com

## Options

```
  Usage: isitup [-v] [-h] domain
  -h, --help     Show help
  -v, --version  Show isitup-cli's version
```
